## Application Overview
ProCrastinator is a VR productivity application built for Android phones using the Google VR SDK in Unity. It provides a focused writing and task-management environment designed to help users overcome procrastination through an immersive virtual workspace.

## How to Build & Run

### Prerequisites
- Unity 2021 LTS or newer with Android Build Support module installed
- Android SDK and JDK configured in Unity Preferences
- Google VR SDK imported into the project
- A Google Cardboard-compatible Android phone

### Build Steps
1. Clone the repository: `git clone https://github.com/gaelxxl34/The-Pro-Crastinators-Team-5`
2. Open the project in Unity.
3. Navigate to **File > Build Settings** and select **Android**.
4. Click **Switch Platform**, then open the ProCrastinator scene.
5. Click **Build** and export as `preliminary.apk`.
6. Transfer the APK to your Android device and install it.

## Interaction Techniques
The application is controlled using Android phone controller input (Google Cardboard button):

| Input | Action |
|---|---|
| Head Gaze (Reticle) | Aim the center reticle at objects or UI elements to select them |
| Cardboard Button Click | Confirm selection / trigger interaction |
| Left Joystick / D-pad Up-Down | Navigate menus (teleport destination, settings) |
| B / Joystick Button | Confirm menu selection, activate lever, toggle display |
| O / U Button | Open / close settings menu |

## Git Rules

Do **not** include these folders/files in your commits — they are auto-generated by Unity:

- `Library/`, `Temp/`, `Obj/`, `Build/`, `Logs/`, `UserSettings/`
- `.vs/`, `.vscode/`, `.idea/`
- `*.csproj`, `*.sln`, `*.slnx`, `*.suo`, `*.approj`
- `.DS_Store`, `Thumbs.db`

Only commit:
- `Assets/` — scripts, scenes, prefabs, etc.
- `ProjectSettings/` — shared project settings

- `Packages/manifest.json` and `Packages/packages-lock.json`

## Task Assignments

### Kevin — Keyboard Data Analytics Feature
- Implement a keyboard data analytics system that tracks and displays typing statistics
- Collect metrics such as keys pressed, typing speed (WPM), accuracy, and key frequency
- Provide a summary/dashboard of the analytics data to the user

### Rut — Time Management Tool
- Build a creative time management tool that lets the user set a session timer
- User can input the time in **seconds** or **minutes**
- While the timer is running, the writer can type normally
- When the set time expires:
  - **Block writing** — disable the ability for the writer to continue typing
  - **Show a countdown** to exit the game
- The experience should feel fast-paced and engaging

### Gael — Advanced Teleportation & Menus
- Implement advanced teleportation mechanics for navigating the VR environment
- Design and build in-game menus (main menu, pause menu, settings, etc.)
- Ensure smooth UX transitions between teleportation and menu interactions
